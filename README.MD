# 📧 Classificador de E-mails com IA (Llama 3.3)

Este projeto é uma ferramenta de triagem automatizada para e-mails corporativos. Utilizando inteligência artificial de última geração, o sistema classifica mensagens entre **Produtivas** e **Improdutivas**, extraindo justificativas e sugerindo respostas automáticas de forma instantânea.

---

## 🌟 Funcionalidades

- **Entrada Versátil**: Suporte para texto colado manualmente ou upload de arquivos (**PDF** e **TXT**).
- **IA de Alta Performance**: Processamento realizado via **Groq Cloud** utilizando o modelo **Llama 3.3 70B**, garantindo respostas em milissegundos.
- **Extração de Texto**: Leitura automática de arquivos PDF através da biblioteca `pypdf`.
- **Interface Intuitiva**: UI moderna construída com **Tailwind CSS**, com sistema de feedback visual para o usuário.
- **Botão de Limpeza**: Funcionalidade para resetar rapidamente a análise e iniciar um novo processamento.

---

## 🛠️ Tecnologias

- **Backend**: Python 3.10+
- **Framework Web**: Flask
- **IA/ML**: API Groq (Llama 3.3 70B)
- **Document Parsing**: pyPDF
- **Frontend**: HTML5, JavaScript (ES6+), Tailwind CSS
- **WSGI Server**: Gunicorn (Produção)
